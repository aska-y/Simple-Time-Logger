## Simple Time Loggerについて
<p>
複数のプロジェクトを並行して進める際に、プロジェクトごとに実際に作業した工数が記録・管理できるものがあると便利だと思い、Webアプリを作成しました。<br />
プロジェクト名と概要、日々の工数、トータル作業時間がわかる内容で、シンプルに仕上げました。<br />
異なる環境からでも記録できるように Webアプリにすることにすべく、React & TypeScript に挑戦しました。<br />
<br />
DjangoでDBとAPIを構成し、Reactから呼び出して表示させています。 <br />
ストップウォッチを開始後、ストップボタン押下でUIに即座に反映させるとともに、AxiosdでDBに記録させています。<br />
作業日、時間、合計時間を一覧で表示させ、業務ごとの工数記・管理に役立てる想定です。<br />
また、完了/未完了でプロジェクト一覧を分けて表示させ、プロジェクトを俯瞰して見れるようにしました。　　<br />
</p>
